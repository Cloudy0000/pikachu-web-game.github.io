# pikachu-web-game.github.io
Игра выполнена с помощью ИИ от Google за 3 часа

# Функциональные возможности:
- Управление персонажем через клавиши       
Вправо: →        
Влево: ←        
Прыжок: ↑ или Space

- Плавная анимация движения
- Игровая физика       
Гравитация       
Коллизии с платформами         
Обработка столкновений       
- Игровой интерфейс        
Отображение количества собранных ягод        
Кнопки управления        
- Игровая логика        
Система подсчета собранных ягод         
Условие победы - сбор всех ягод
# Инструкция по запуску
Для старта достаточно открыть страницу        
Для рестарта нужно нажать F5
Для победы нужно собрать все ягоды
